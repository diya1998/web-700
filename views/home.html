Intro to the website:
This website has : * three static views i.e home ,about ,homeDemo . * Five routes that supply data i.e All Students,Students by Course, A Single Student By Student Number, All TAs and All Courses. All realted relevant links to pages are as bellow :
  1. Home page using GET "/" request with url "http://localhost:8080".
  2. About page using GET "/about" request with url "http://localhost:8080/about".
  3. Demo page using by GET "/htmlDemo" request with url "http://localhost:8080/htmlDemo".
  4. GET "/students" request with url "http://localhost:8080/students" will return a json string containing all of the students coming from the getAllStudents function.
  5. GET "/students?course=value" request with url "http://localhost:8080/students?course=4" will return a json string consisting of all students where value could be one of 1, 2, 3, to 7 (as there are 7 cources in the dataset)" - using the getStudentsByCourse(course) function in the collegeData
  6. GET "/tas" request with url "http://localhost:8080/tas" again returns a json format string containing all of the tas output from the getTAs function ( If any error in promise run the object back as json {message:"no results"} )
  7. GET "/courses" request with url "http://localhost:8080/courses" will return a json format string with all of the courses output from the getCourses function
  8. GET "/students/:num" request from url "http://localhost:8080/students/2" will return a json format string with only a single student where studentNum property matches the num parameter in the route i.e. Student with ID 5 in the example case.

Intro On HTML5

HTML5 is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and final major HTML version that is a World Wide Web Consortium (W3C) recommendation. The current specification is known as the HTML Living Standard. It is maintained by the Web Hypertext Application Technology Working Group (WHATWG),a consortium of the major browser vendors (Apple, Google, Mozilla, and Microsoft).
